# Mobile Breakdown Cards

When a desktop tooltip is too large for the viewport, split the behavior:

- keep the floating tooltip on desktop
- render a compact breakdown card below the chart on mobile

## Good mobile card contents

- current x-axis position or label
- short list of active series values
- a total line
- active-series emphasis

## Good mobile card behavior

- update from touch or chart selection, not hover alone
- stay within the screen width
- avoid covering the plot
- remain visually connected to the chart through color and typography
