# Donut Legend Compaction

Donut and pie legends often fail on mobile because the default side legend is taller than the chart.

## Preferred pattern

- hide the tall side legend on mobile
- move the legend below the donut
- use a compact grid or wrapped chip list
- shorten long labels when needed
- include the most useful secondary number directly in the legend item, often the percent

## Result

The chart and legend read as one module and can fit inside the viewport without horizontal scroll.
