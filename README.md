# SKILLS

Public, sanitized Codex skills for analysis, reuse, and review.

This repository contains only generalized skill packages that were reviewed to avoid:

- corporate hostnames and internal infrastructure details
- local private paths and usernames
- keys, secrets, or deployment credentials
- project-specific sensitive context

Current public skills:

- `skills/playwright`
- `skills/recharts-stacked-interactivity`
- `skills/jump-host-static-deploy`

More skills can be added over time after the same sanitization pass.
